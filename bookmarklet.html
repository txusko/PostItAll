Post It All! Bookmarklet

Add a post it to any webpage, and recover it later with Post it all!
Just add the bookmarklet in your bookmark toolbar and start "Postit-ing" all!!

Post It All!

It's so easy

  1. Install it by adding the bookmarklet to your bookmarks toolbar.
  2. Press the "Post It All!" bookmark
  3. Add post it's to your page pressing the "Add postit" button at the left of the window
  4. Recover it later by pressing the "Post It All!" bookmark

No login required

This service is anonymously and all the stored data is only stored in your local web browser. This is an advantage because all the data you store, is only accessible by you, locally in this web browser / machine, and a disadvantage because this is not a cloud storage service, so, your data will be not recoverable by other browsers or machines.

Remember anything

You can add as many post it's as you want. You can write annotations, remembers, to do's list, or whatever you imagine in any web page you visit. Later, you can recover all the stuff when you needed, only by pressing the "Post It All!" bookmarklet.

How it works

Install

Drag and drop the green botton below to your bookmarks toolbar

Use it

When you want to add an annotation or you want to recover an old note from the site your are visiting, just press the "Post It All!" link you have already added to your bookmark toolbar. If the library load correctly you will see a fixed blue button in the middle-left of your browser window.

Internal storage

All the data you add to a note is stored locally throw HTML5 Web Storage.

Features

  • Post it features Change background-color, text-color, resize it, move it ...
  • Autosave All data-notes saved automatically on before unload event.
  • Add Postit button Just press it if you want a sticky note floating next to the button. (try to select some text before)
  • Hotkey Create a postit in your mouse position by pressing: "Caps + mouse click". (try to select some text before)

Limits & problems

In Chrome browser, the Content Security Policy restrict the use of this bookmarklet in some pages such like google. I'm currently working in the "Post It All! Chrome Extension", that will not be limited, and will be autoloaded in the begining of each page.
Anyway, if you want to load Insecure content, you need to follow this steps: "This page has insecure content."

Host your own version

Edit /bookmarklet/postitall.js and change PIAurl by your server PostItAll url (Ex: http://localhost/PostItAll/). After this change, all the external loaded libraries (jQuery, jQuery UI, CSS, Minicolors plugin & PostItAll plugin) will be called and loaded from your site PostItAll directory.
If you don't change this, the bookmarklet will try to load the extra content from the default PIAurl : https://googledrive.com/host/0B2vKerMpRnudUGJlWC1yN3VUZWs/
