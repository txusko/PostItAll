About Post It All. What, who, how and why.

"Post It All!", is just another sticky notes javascript plugin for the web.

You can add it to your site as a jQuery plugin, or use the Bookmarklet to add sticky notes for a later recovery.

It was developed in my spare time just for fun. You can check my entery profile here: Javi Filella

Current Features

jQuey plugin:
  • Drag & Drop your post it's to the position you chose in the page.
  • Autoheight the height of the postit will automaticly feet to the content.
  • Resize the width of your post it's and feet better to the content. jQuery UI dependancy
  • Savable you can save the postit in Local Storage.
  • Personalize every postit with diferent background colors, text color and text shadow.
  • Easy color-change interface throw Minicolors Plugin. Minicolors depandecy
Bookmarklet:

The data is saved throw HTML5 Web Storage locally. So, all the data you store, is only accesible in this machine and in this browser.

  • Autosave Save all the post it's content data locally in the browser for a later recovery. Just press again the button when you want to recover all your stuff.
  • Create a postit with the "Add postit" button in the middle-left side of the page (try to select some text before)
  • Hotkey Create a postit in your mouse position by pressing: "Caps + mouse click" (try to select some text before)

Limits & problems

  • Post It All will not work in old browsers. For the Bookmarklet, ensure your browser have support for the HTML5 Web Storage technology
  • Note that for the "autoheight" feature, the sticky will try to resize to the content while you type in. In some cases (for example when you paste html content), this could result in a strange behaviour. Try to resize it, or to add an extra space at the end of the note to forze the autoresize.

Note While I was developing "Post It All!", I discovered tons of similar plugins, service and extensions for browsers that make the same work or have similar characteristics as "Post It All!". Be free to check it out and use the one who fit your needs.
