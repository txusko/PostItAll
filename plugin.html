Post It All! jQuery plugin

A jQuery plugin to add postits to your webpage

Download plugin

Examples

Basic example

This is a basic example, where you can convert the content of an HTML object into an editable, draggable and resizable postit.

Code

// HTML
<div id="idAddDemoPostit">Content of the postit 1</div>
<div class="classAddDemoPostit">Content of the postit 2</div>
<div class="classAddDemoPostit">Content of the postit 3</div>
<div class="classAddDemoPostit">Content of the postit 4</div>

//JAVASCRIPT
<script>
$(document).ready(function() {
  $('#idAddDemoPostit').postitall();
  $('.classAddDemoPostit').postitall();
});
</script>

Form example

In this example, we take the parameters from form inputs to create a new postit. With the 'newPostit' parameter set to true, you can create as many post it's as you want.

Code

// HTML
<form>
<div>
  <label for="idBgTextColor">Background-Color:</label>
  <input id="idBgTextColor" type="text" name="bgColor" value="#0000ff" class="minicolors">
</div>
<br>
<div>
  <label for="idTextColor">Color:</label>
  <input id="idTextColor" type="text" name="textColor" value="#333333" class="minicolors">
</div>
<br>
<div>
  <label for="idContent">Content:</label>
  <textarea id="idContent" name="textColor">This is another demo</textarea>
</div>
<div>
  <input type="button" class="btn btn-primary" value="Add this as a postit" id="idAddDemoPostit2">
</div>
<div id="idAddDemoPostit2_result"></div>
</form>

// JAVASCRIPT
<script>
$(document).ready(function() {
  $('#idAddDemoPostit2').click(function() {
      $('#idRight').postitall({
        'textcolor'       : $('#idTextColor').val(),
        'backgroundcolor' : $('#idBgTextColor').val(),
        'description'     : $('#idContent').val(),
        'newPostit'       : true
      });
  });
});
</script>

Recover options

You can recover the options (position, dimensions, content, ...) of a postit by calling the 'options' method

Code

// HTML
<p><input id="idRecoverButton" type="button" class="btn" value="Recover all postit options"></p>
<textarea id="idRecoverOptions" style="width:100%;height:350px;">
</textarea>

// JAVASCRIPT
<script>
$('#idRecoverButton').click(function() {
  $('.PIApostit').each(function() {
    var val = $('#idRecoverOptions').val();
    $('#idRecoverOptions').val(val+'\n'
      +JSON.stringify($(this).postitall('options'))
      +'\n');
  });
});
</script>

Documentation

Installation

Add the css and the js as allways:

<link rel="stylesheet" href="plugin/jquery.postitall.css">
<script src="plugin/jquery.postitall.js"></script>
Recomendations:
If you want the resize feature for the note, you should add the jQuery UI library: 
<link rel="stylesheet" href="plugin/jquery-ui.css">
<script src="plugin/jquery-ui-1.10.1.min.js"></script>
If you want the friendly color-plugin choser, you should add the Minicolors jQuery Plugin: 
<link rel="stylesheet" href="plugin/jquery.minicolors.css">
<script src="plugin/jquery.minicolors.js"></script>

Properties

Property Explain
id Id of the note. Each note will have an unique identifier.
Default number of notes + 1
created Creation date of the note (not used)
Default Date.now()
domain Domain where the note was created (not used)
Default window.location.origin
page Page where the note was created (not used)
Default window.location.pathname
backgroundcolor Background color of the note
Default #FFFC7F
textcolor Color of the text
Default #333333
textshadow Specify if the text have a white shadow or not
Default true
position Type of position for the note.
  • absolute: set the position in the page where the top left corner of your page is 0 for posX and posY.
  • relative: set the position relative to the container element where the top left corner of the container is 0 for posX ans posY.
    Note that the plugin convert the relative to absolute position for preventing the "preservation of height" of the note in the page layout making them float.
Default relative
posX Number of pixels in the horizontal axis, starting from the top of the container (see position), where the postit will be drawn.
Default 5px
posY Number of pixels in the vertical axis, starting from the left of the container (see position), where the postit will be drawn.
Default 5px
height Height of the note.
Remember that this height will be increased in 30 pixels for the toolbar space of the note.
Default 180
width Width of the note
Default 200
minHeight Resizable min-height
Default 152
minWidth Resizable min-width
Default 131
description Content of the note. You can add html too.
Default blank
newPostit It will create a new note inside the selector given.
This note will have the newPostIt_{id} as identifier
Default false
autoheight Set autoheight feature on or off
Default true
draggable Set draggable feature on or off
Default true
resizable Set resizable feature on or off
Default true
removable Show or hide the "trash" icon
Default true
changeoptions Show or hide the "options" icon
Default true
savable Saves post it in Local Storage
Default false
Note that You can recover all the notes with the .PIApostit selector

Full example with all properties set to the defaults.

<script>
$('#some_id').postitall({
	id              : 0,                        //Id
	created         : Date.now(),               //Creation date
	domain          : window.location.origin,   //Domain in the url
	page            : window.location.pathname, //Page in the url
	backgroundcolor : '#FFFC7F',                //Background color
	textcolor       : '#333333',                //Text color
	textshadow      : true,                     //Shadow in the text
	position        : 'relative',               //Position absolute or relative
	posX            : '5px',                    //top position
	posY            : '5px',                    //left position
	height          : 180,                      //height
	width           : 200,                      //width
	minHeight       : 152,                      //resizable min-width
	minWidth        : 131,                      //resizable min-height
	description     : '',                       //content
	newPostit       : false,                    //create a new postit
	autoheight      : true,                     //Set autoheight feature on or off
	draggable       : true,                     //Set draggable feature on or off
	resizable       : true,                     //Set resizable feature on or off
	removable       : true,                     //Set removable feature on or off
	changeoptions   : true,                     //Set options feature on or off
	savable         : false,                    //Save postit in local storage
});
</script>
